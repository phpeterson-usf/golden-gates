MultibaseNumberInput Test

To test the fix:

  1. Run the app with npm run dev
  2. Add an Input component to the canvas
  3. Select it to open the inspector
  4. In the Value field, type "0xa9ff"
  5. Press Tab to blur the field
  6. The field should still show "0xa9ff" (not reset to "0")

The fix moves the lastEmittedValue and lastEmittedBase variables to the proper scope so they persist between function calls.
